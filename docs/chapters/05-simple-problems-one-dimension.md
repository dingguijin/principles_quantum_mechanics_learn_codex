# 5. Simple Problems in One Dimension

这一章开始解具体问题。一维问题是量子力学的训练场：自由粒子、势垒、势阱和束缚态会反复出现。

## 本章目标

- 会写一维定态薛定谔方程。
- 理解边界条件和归一化条件。
- 区分束缚态和散射态。
- 理解能级量子化如何从边界条件出现。

## 核心讲解

一维哈密顿算符通常写成：

```text
H = p^2/(2m) + V(x)
```

在位置表象中：

```text
p -> -i hbar d/dx
```

所以定态薛定谔方程变成：

```text
[-hbar^2/(2m) d^2/dx^2 + V(x)] psi(x) = E psi(x)
```

这本质上是一个本征值问题：找出允许的能量 `E` 和对应波函数 `psi(x)`。

量子化常常不是硬规定，而是由边界条件逼出来的。比如无限深势阱中，波函数在边界必须为零，只允许某些波长，因此只允许某些能量。

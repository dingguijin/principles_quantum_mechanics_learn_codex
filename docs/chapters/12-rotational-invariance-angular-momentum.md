# 12. Rotational Invariance and Angular Momentum

角动量是三维量子力学的核心。本章从转动对称性进入角动量代数。

## 本章目标

- 理解转动不变性。
- 知道角动量算符的对易关系。
- 区分 `L^2` 和 `L_z`。
- 理解角动量量子数。

## 核心讲解

经典轨道角动量为：

```text
L = r x p
```

量子力学中，角动量分量成为算符，并满足非平凡对易关系。一个重要结果是：不同方向的角动量分量不能同时任意精确确定。

常同时讨论：

```text
L^2
L_z
```

它们可以有共同本征态，常用量子数 `l` 和 `m` 标记。

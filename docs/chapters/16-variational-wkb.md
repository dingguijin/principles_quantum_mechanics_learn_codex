# 16. The Variational and WKB Methods

很多量子问题不能精确求解。本章介绍两种重要近似：变分法和 WKB。

## 本章目标

- 理解近似方法的必要性。
- 掌握变分法的基本思想。
- 知道 WKB 适合缓慢变化势场。
- 理解近似的适用范围比公式本身更重要。

## 核心讲解

变分法的核心是：用一个带参数的试探波函数估计基态能量。若试探态归一化，则：

```text
E_trial = <psi|H|psi>
```

它给出基态能量的上界。通过调参数让 `E_trial` 尽量小，就得到较好的近似。

WKB 方法适合势能变化相对缓慢、波长变化不剧烈的情形。它把波函数写成带快速相位的形式，从而连接经典动量和量子相位。

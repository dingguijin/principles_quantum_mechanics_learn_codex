# 18. Time-Dependent Perturbation Theory

含时微扰研究外部作用随时间变化时，系统如何在不同能级之间跃迁。

## 本章目标

- 理解跃迁振幅和跃迁概率。
- 知道含时微扰和定态微扰的区别。
- 认识共振条件。
- 理解 Fermi 黄金规则的用途。

## 核心讲解

若哈密顿量含有随时间变化的小项：

```text
H(t) = H0 + V(t)
```

即使系统一开始在 `H0` 的某个本征态中，随时间也可能跃迁到其他本征态。

含时微扰理论计算的核心是：

```text
初态 -> 末态 的跃迁振幅
```

当外部扰动频率匹配两个能级差时，跃迁会显著增强，这就是共振思想。

# 20. The Dirac Equation

Dirac 方程把量子力学和狭义相对论结合起来，并自然引出反粒子和自旋结构。

## 本章目标

- 理解非相对论薛定谔方程的局限。
- 知道相对论能量关系。
- 理解 Dirac 方程为什么需要多分量波函数。
- 初步认识反粒子思想。

## 核心讲解

非相对论能量为：

```text
E = p^2/(2m)
```

相对论能量满足：

```text
E^2 = p^2 c^2 + m^2 c^4
```

直接把这个关系量子化会遇到解释困难。Dirac 的思路是寻找一个对时间和空间导数都一阶的方程。这样做需要引入满足特定代数关系的矩阵，波函数也变成多分量对象。

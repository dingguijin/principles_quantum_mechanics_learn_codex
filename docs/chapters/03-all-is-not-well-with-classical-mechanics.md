# 3. All Is Not Well with Classical Mechanics

本章的作用是说明：经典力学不是错在数学不漂亮，而是错在某些实验事实无法解释。量子力学的出现是被实验逼出来的。

## 本章目标

- 知道经典力学在哪些地方失效。
- 理解黑体辐射、光电效应、原子光谱等现象的共同指向。
- 接受“状态不能总被经典轨迹描述”的必要性。

## 核心讲解

经典图像里，粒子有确定的位置和动量，场可以连续吸收或发出能量。但微观实验显示：

- 能量交换经常呈现离散性。
- 原子光谱不是连续的，而是有特定谱线。
- 光有波动性，也表现出粒子性。
- 电子有粒子性，也能产生干涉。

这些现象说明，微观世界不能简单套用“点粒子沿确定轨迹运动”的图像。

这一章不是要你记很多实验细节，而是要建立一个判断：量子力学的奇怪规则不是任意发明，它们是为了同时解释概率、干涉、离散能级和测量结果。

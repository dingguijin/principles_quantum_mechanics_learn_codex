# 6. The Classical Limit

量子力学必须在适当条件下回到经典力学。本章研究这个连接：什么时候量子结果看起来像经典轨迹？

## 本章目标

- 理解经典极限不是简单令 `hbar = 0`。
- 知道波包和期望值如何接近经典运动。
- 理解 Ehrenfest 思想：期望值在某些条件下 obey 类似经典的运动方程。

## 核心讲解

量子态一般不是一个点，而是分布。经典力学说粒子在某个位置；量子力学给出位置概率分布。

如果一个波包足够集中，并且势能在波包宽度内变化不剧烈，那么位置和动量的期望值可能近似满足经典方程。

经典极限常常需要同时考虑：

- 系统尺度远大于量子尺度。
- 相位快速振荡导致干涉平均掉。
- 测量精度无法分辨微观涨落。

4. 经典极限和测量精度有什么关系？

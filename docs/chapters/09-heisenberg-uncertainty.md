# 9. The Heisenberg Uncertainty Relations

不确定性关系不是测量仪器不够精密，而是量子态本身的结构限制。它来自非对易算符。

## 本章目标

- 理解方差和不确定度。
- 知道位置和动量不确定性关系。
- 理解不确定性与对易子的关系。
- 避免把不确定性误解为实验误差。

## 核心讲解

对观测量 `A`，不确定度可以理解为测量结果围绕平均值的分散程度：

```text
(Delta A)^2 = <A^2> - <A>^2
```

位置和动量满足：

```text
Delta x Delta p >= hbar/2
```

更一般地，不确定性与对易子有关。如果两个算符不对易，就通常不能同时拥有任意精确的确定值。

# 14. Spin

自旋是量子系统的内禀角动量，不是小球真的在自转。本章会让“抽象态空间”的意义变得更清楚。

## 本章目标

- 理解自旋不是经典转动。
- 掌握自旋 `1/2` 的两维态空间。
- 认识 Pauli 矩阵。
- 理解 Stern-Gerlach 类实验的意义。

## 核心讲解

自旋 `1/2` 系统的态可写成两分量向量：

```text
|psi> = a |up> + b |down>
```

其中：

```text
|a|^2 + |b|^2 = 1
```

测量某方向自旋时，只能得到两个结果。不同方向的自旋测量对应不同算符，因此同一个态在不同测量方向下会有不同概率分布。

# 2. Review of Classical Mechanics

量子力学不是凭空替代经典力学，而是在经典力学的语言上重新解释“状态、动量、能量、演化”。本章要把牛顿、拉格朗日和哈密顿三种说法连起来。

## 本章目标

- 知道位置、速度、动量、能量的含义。
- 理解拉格朗日量 `L = T - V`。
- 理解哈密顿量 `H = T + V` 在保守系统中常对应总能量。
- 知道广义坐标和正则动量。
- 为后面的正则量子化做准备。

## 核心讲解

牛顿力学常写成：

```text
F = ma
```

它直接描述力如何改变运动。拉格朗日力学换了一个角度：不直接从力出发，而从动能 `T` 和势能 `V` 出发：

```text
L = T - V
```

系统的真实路径满足作用量取驻值：

```text
S = integral L dt
```

哈密顿力学进一步把变量换成坐标 `q` 和动量 `p`，用哈密顿量 `H(q,p)` 描述演化：

```text
dq/dt = partial H / partial p
dp/dt = - partial H / partial q
```

量子力学最常见的入口就是把经典变量 `q, p` 提升为算符，并让它们满足特定对易关系。

## 为什么要学这一章

如果不懂哈密顿量，薛定谔方程会显得像凭空出现：

```text
i hbar d|psi>/dt = H |psi>
```

这里的 `H` 不是随便来的，它继承了经典力学中“决定系统演化”的角色。

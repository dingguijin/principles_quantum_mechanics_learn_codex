# 4. The Postulates: A General Discussion

这一章正式给出量子力学的规则。前面学的线性代数在这里变成物理语言：态、观测量、测量、概率和时间演化。

## 本章目标

- 理解态向量表示系统状态。
- 理解可观测量由厄米算符表示。
- 理解测量结果、本征值和本征态的关系。
- 会用展开系数计算测量概率。
- 理解薛定谔方程的角色。

## 核心讲解

量子力学的基本结构可以压缩成几句话：

| 对象 | 量子力学写法 |
| --- | --- |
| 态 | $`\lvert \psi\rangle`$ |
| 观测量 | 厄米算符 $`A`$ |
| 可能结果 | $`A`$ 的本征值 |
| 概率 | 态在相应本征态方向上的投影模平方 |
| 演化 | $`i\hbar\,\frac{d}{dt}\lvert\psi\rangle = H\lvert\psi\rangle`$ |

若：

$$
\lvert \psi \rangle = \sum_{n} c_{n} \lvert a_{n} \rangle
$$

其中 $`\lvert a_n\rangle`$ 是观测量 $`A`$ 的本征态，那么测量得到 $`a_n`$ 的概率是：

$$
P(a_{n}) = \lvert c_{n}\rvert^2
$$

测量不是单纯“读取已有属性”。在量子理论中，测量会把态和被测算符的本征态联系起来。

## 学习重点

这一章要反复对照第一章。你会发现：

- 本征值不再只是线性代数概念，而是测量结果。
- 正交归一不再只是计算方便，而是互斥结果的结构。
- 幺正演化不再只是保持长度，而是保持总概率。

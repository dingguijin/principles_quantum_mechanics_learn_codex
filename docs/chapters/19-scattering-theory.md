# 19. Scattering Theory

散射理论研究粒子遇到势场或靶后如何偏转。它把可观测的截面和量子振幅联系起来。

## 本章目标

- 理解入射波、散射波和散射振幅。
- 知道微分截面的物理意义。
- 初步掌握 Born 近似思想。
- 理解相移在中心势散射中的作用。

## 核心讲解

散射问题通常不是问束缚能级，而是问：

```text
入射粒子从某方向来，最后往各方向出去的概率分布是什么？
```

远离散射中心时，波函数常分解为入射部分和向外传播的散射部分。散射振幅决定不同方向的强度，截面把理论结果转成实验可测量的量。

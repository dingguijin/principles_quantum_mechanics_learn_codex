# 1. Mathematical Introduction

这一章不是普通的数学复习，而是在建立量子力学的语言。你以后看到的态、波函数、测量、能量、自旋、角动量，本质上都在使用本章的几个工具：向量空间、内积、算符、本征值、基底变换。

如果你是零基础，请不要试图“一遍读懂”。这章建议分 6 次学习，每次只掌握一组概念。

## 本章学习路线

1. 第 1 课：为什么量子态要当作向量。
2. 第 2 课：基底、坐标和展开。
3. 第 3 课：内积、长度、正交和归一化。
4. 第 4 课：线性算符和矩阵。
5. 第 5 课：本征值、本征向量和测量预告。
6. 第 6 课：厄米、幺正、投影以及无限维推广。

学习目标不是背定义，而是能把符号翻译成普通语言。比如：

```text
A|psi> = |phi>
```

你应该能读成：

```text
算符 A 作用在状态 |psi> 上，得到新状态 |phi>
```

## 第 1 课：为什么量子态要当作向量

### 1.1 从一个最小实验想起

先想一个只有两个可能结果的系统。比如某个量子系统测量后只可能得到：

```text
结果 1
结果 2
```

我们暂时不关心它是什么具体系统。只问：怎样描述它的状态？

最简单的办法是给两个结果各配一个“权重”：

```text
状态 = (c1, c2)
```

这里 `c1` 和 `c2` 不是概率，而是概率振幅。真正概率是：

```text
P1 = |c1|^2
P2 = |c2|^2
```

如果系统只可能出现这两个结果，那么：

```text
|c1|^2 + |c2|^2 = 1
```

你已经看到向量语言自然出现了：一个状态可以用一组分量表示。

### 1.2 什么是向量空间

一个向量空间是一类对象的集合，里面的对象可以相加，也可以乘以数。

例如：

```text
v = (1, 2)
w = (3, -1)
```

相加：

```text
v + w = (4, 1)
```

数乘：

```text
2v = (2, 4)
```

量子态也要允许相加和数乘，因为量子力学有叠加原理。如果 `|1>` 和 `|2>` 是两个可能状态，那么：

```text
|psi> = c1|1> + c2|2>
```

也是一个可能状态。

这句话就是量子力学形式体系的入口。它不是说系统“一半在状态 1，一半在状态 2”，而是说系统处在两个基态的叠加态，测量概率由系数决定。

### 1.3 Dirac 记号先怎么读

书里会大量使用：

```text
|psi>
```

这叫 ket，读作“ket psi”。你现在不用管名字来源，只要知道它表示一个向量，也就是一个量子态。

对应还有：

```text
<psi|
```

这叫 bra。bra 和 ket 合起来：

```text
<phi|psi>
```

表示两个态的内积。它会告诉我们两个态有多“重合”。

先记一个翻译表：

| 符号 | 普通语言 |
| --- | --- |
| `|psi>` | 一个态向量 |
| `<psi|` | 与 `|psi>` 对应的 bra |
| `<phi|psi>` | `|phi>` 与 `|psi>` 的内积 |
| `A|psi>` | 算符 `A` 作用在态上 |
| `|psi> = c1|1> + c2|2>` | 态在一组基底上的展开 |

### 1.4 最小计算例子

设：

```text
|1> = (1, 0)
|2> = (0, 1)
```

如果：

```text
|psi> = (3/5)|1> + (4/5)|2>
```

那么在这组基底下：

```text
|psi> = (3/5, 4/5)
```

概率是：

```text
P1 = |3/5|^2 = 9/25
P2 = |4/5|^2 = 16/25
P1 + P2 = 1
```

所以这是归一化态。

## 第 2 课：基底、坐标和展开

### 2.1 基底是什么

基底是一组“基本方向”。有了基底，就可以用坐标描述向量。

二维平面最常用基底是：

```text
e1 = (1, 0)
e2 = (0, 1)
```

任何二维向量都能写成：

```text
v = v1 e1 + v2 e2
```

比如：

```text
(3, 4) = 3(1, 0) + 4(0, 1)
```

这里 `(3,4)` 是坐标表示，`e1` 和 `e2` 是基底向量。

### 2.2 向量本身和坐标不是一回事

这是初学者最容易混淆的地方。

一个向量是一个对象。坐标是你选定某组基底后，用数字描述它的方式。

换一组基底，同一个向量的坐标会变。向量没有变，描述方式变了。

类比：同一个地点可以用中文地址表示，也可以用经纬度表示。地点没变，坐标系统变了。

量子力学中也一样。同一个态可以用位置表示：

```text
psi(x)
```

也可以用动量表示：

```text
phi(p)
```

它们不是两个不同物理系统，而是同一个态在不同基底下的表示。

### 2.3 展开系数是什么意思

如果：

```text
|psi> = c1|1> + c2|2> + c3|3>
```

那么 `c1, c2, c3` 是 `|psi>` 在这三个基底方向上的分量。

若这组基底是正交归一的，那么：

```text
|c1|^2, |c2|^2, |c3|^2
```

常常可以直接解释为对应测量结果的概率。

但注意：这句话依赖于你选的基底对应某个观测量的本征态。第四章会正式讲。

### 2.4 最小计算例子

设：

```text
|a> = (1, 0)
|b> = (0, 1)
|psi> = (2, -1)
```

那么：

```text
|psi> = 2|a> - 1|b>
```

如果要归一化它，先算长度：

```text
||psi|| = sqrt(2^2 + (-1)^2) = sqrt(5)
```

归一化态是：

```text
|psi_norm> = (1/sqrt(5))(2, -1)
           = (2/sqrt(5), -1/sqrt(5))
```

此时两个方向上的概率是：

```text
P_a = |2/sqrt(5)|^2 = 4/5
P_b = |-1/sqrt(5)|^2 = 1/5
```

## 第 3 课：内积、长度、正交和归一化

### 3.1 内积先理解成“重合程度”

两个向量的内积告诉你它们有多对齐。

实向量例子：

```text
v = (1, 0)
w = (0, 1)
```

内积：

```text
v · w = 1 x 0 + 0 x 1 = 0
```

它们互相垂直，所以内积为 0。

再看：

```text
u = (1, 0)
v = (3, 0)
```

内积：

```text
u · v = 3
```

它们方向完全一致，所以内积非零。

### 3.2 复向量内积为什么要复共轭

量子态通常有复数分量。复向量内积定义为：

```text
<v|w> = v1* w1 + v2* w2 + ...
```

为什么第一个向量要取复共轭？因为我们希望长度平方：

```text
<v|v>
```

一定是非负实数。

例子：

```text
v = (1, i)
```

如果不用复共轭，得到：

```text
1 x 1 + i x i = 1 - 1 = 0
```

这会荒谬地说非零向量长度为 0。

用复共轭：

```text
<v|v> = 1* x 1 + i* x i = 1 + (-i)i = 2
```

这才是合理的长度平方。

### 3.3 长度和归一化

长度定义为：

```text
||v|| = sqrt(<v|v>)
```

量子态要求：

```text
<psi|psi> = 1
```

这叫归一化。物理意义是总概率为 1。

如果一个态还没有归一化，比如：

```text
v = (1, i)
```

长度平方：

```text
<v|v> = 2
```

长度：

```text
||v|| = sqrt(2)
```

归一化：

```text
|psi> = (1/sqrt(2))(1, i)
```

检查：

```text
<psi|psi> = 1/2 + 1/2 = 1
```

### 3.4 正交

如果：

```text
<v|w> = 0
```

则 `v` 和 `w` 正交。

在量子力学中，正交常常表示两个状态可以被清楚区分。比如某个测量的两个不同结果对应两个正交本征态。

### 3.5 正交归一基底

一组基底若满足：

```text
<i|j> = 0  当 i != j
<i|i> = 1
```

可合写成：

```text
<i|j> = delta_ij
```

这里 `delta_ij` 是 Kronecker delta：

```text
delta_ij = 1  当 i = j
delta_ij = 0  当 i != j
```

正交归一基底最大的好处是：展开系数可以用内积直接取出来。

如果：

```text
|psi> = c1|1> + c2|2>
```

那么：

```text
<1|psi> = c1
<2|psi> = c2
```

## 第 4 课：线性算符和矩阵

### 4.1 算符是什么

算符是作用在向量上的规则。

写作：

```text
A|psi> = |phi>
```

普通语言：

```text
A 把状态 |psi> 变成状态 |phi>
```

在有限维空间里，算符可以用矩阵表示。

### 4.2 线性是什么意思

线性算符满足：

```text
A(a|v> + b|w>) = aA|v> + bA|w>
```

这句话很重要。它说：如果一个态是叠加态，那么算符对它的作用可以分别作用到每个部分上，再加起来。

例子：

```text
A = [[2, 0],
     [0, 3]]
```

对：

```text
v = (1, 1)
```

作用：

```text
Av = (2, 3)
```

如果：

```text
v = e1 + e2
```

那么：

```text
A(e1 + e2) = Ae1 + Ae2 = (2,0) + (0,3) = (2,3)
```

这就是线性。

### 4.3 矩阵乘法怎么读

设：

```text
A = [[1, 2],
     [3, 4]]

v = (5, 6)
```

则：

```text
Av = (1x5 + 2x6, 3x5 + 4x6)
   = (17, 39)
```

不要把矩阵乘法看成神秘规则。它是在计算新向量每个分量由旧分量如何组合得到。

### 4.4 算符的矩阵表示依赖基底

同一个算符，在不同基底下矩阵可以不同。

这和同一个向量在不同基底下坐标不同是同一件事。

物理上重要的是算符本身做了什么，而不是某个矩阵长什么样。矩阵是选了基底之后的坐标表达。

### 4.5 对易子

两个算符的先后顺序可能影响结果。定义：

```text
[A,B] = AB - BA
```

如果：

```text
[A,B] = 0
```

则说它们对易。

如果不对易，先做 `A` 再做 `B` 和先做 `B` 再做 `A` 不一样。

量子力学中，不对易是非常核心的现象。位置和动量算符不对易，这会导向不确定性关系。

### 最小计算例子

设：

```text
A = [[1, 0],
     [0, -1]]

B = [[0, 1],
     [1, 0]]
```

计算：

```text
AB = [[0, 1],
      [-1, 0]]

BA = [[0, -1],
      [1, 0]]
```

所以：

```text
[A,B] = AB - BA
      = [[0, 2],
         [-2, 0]]
```

它们不对易。

## 第 5 课：本征值和本征向量

### 5.1 先从普通矩阵看

如果：

```text
A|v> = lambda |v>
```

那么 `|v>` 是 `A` 的本征向量，`lambda` 是本征值。

普通语言：

```text
A 作用在 |v> 上以后，没有改变 |v> 的方向，只把它乘了一个数 lambda。
```

例子：

```text
A = [[2, 0],
     [0, 5]]
```

对：

```text
e1 = (1, 0)
```

作用：

```text
Ae1 = (2, 0) = 2e1
```

所以 `e1` 是本征向量，本征值是 `2`。

对：

```text
e2 = (0, 1)
```

作用：

```text
Ae2 = (0, 5) = 5e2
```

所以 `e2` 是本征向量，本征值是 `5`。

### 5.2 为什么量子力学特别重视本征值

量子力学中，可观测量用算符表示。某个可观测量的可能测量结果，对应这个算符的本征值。

这句话先作为预告，第四章会正式讲。

现在你只要记住：

```text
算符 -> 问的问题
本征值 -> 可能答案
本征向量 -> 得到该答案时对应的特殊状态
```

例如，如果能量算符 `H` 满足：

```text
H|E_n> = E_n|E_n>
```

那么 `E_n` 是一个允许的能量值，`|E_n>` 是对应的能量本征态。

### 5.3 怎样手算 2x2 本征值

对矩阵：

```text
B = [[0, 1],
     [1, 0]]
```

找满足：

```text
B(x, y) = lambda(x, y)
```

左边是：

```text
B(x, y) = (y, x)
```

所以：

```text
y = lambda x
x = lambda y
```

把第一式代入第二式：

```text
x = lambda(lambda x) = lambda^2 x
```

若 `x` 不为 0，则：

```text
lambda^2 = 1
lambda = 1 或 -1
```

当 `lambda = 1`：

```text
y = x
```

本征向量可取：

```text
(1, 1)
```

归一化后：

```text
(1/sqrt(2))(1, 1)
```

当 `lambda = -1`：

```text
y = -x
```

本征向量可取：

```text
(1, -1)
```

归一化后：

```text
(1/sqrt(2))(1, -1)
```

### 5.4 本征基底就是“最适合某个问题的基底”

如果你要讨论算符 `A` 对应的测量，那么 `A` 的本征向量基底通常是最自然的基底。

因为在本征态上，算符作用最简单：

```text
A|a_n> = a_n|a_n>
```

一般态可以展开成：

```text
|psi> = sum_n c_n |a_n>
```

测量结果和概率就会围绕这些展开系数组织起来。

## 第 6 课：厄米、幺正、投影和无限维

### 6.1 厄米算符：适合表示可观测量

复矩阵的 dagger 操作表示：

```text
先转置，再复共轭
```

如果：

```text
A^\dagger = A
```

则 `A` 是厄米算符。

例子：

```text
A = [[2, 0],
     [0, 5]]
```

它是厄米的。

再看：

```text
C = [[0, -i],
     [i, 0]]
```

它也是厄米的，因为转置并复共轭后仍回到自己。

厄米算符重要是因为：

1. 本征值是实数。
2. 不同本征值对应的本征向量正交。

物理测量结果必须是实数，所以可观测量要由厄米算符表示。

### 6.2 幺正算符：适合表示保持概率的变换

如果：

```text
U^\dagger U = I
```

则 `U` 是幺正算符。

它保持长度：

```text
<Upsi|Upsi> = <psi|psi>
```

物理意义是：如果一个态总概率原来是 1，经过幺正变换后仍然是 1。

量子态随时间自然演化时，总概率不能丢失，所以时间演化由幺正变换描述。

### 6.3 投影算符：取出某个方向的分量

假设 `|n>` 是归一化向量。投影到 `|n>` 方向的算符是：

```text
P_n = |n><n|
```

对任意态：

```text
P_n|psi> = |n><n|psi>
```

这里：

```text
<n|psi>
```

就是 `|psi>` 在 `|n>` 方向上的分量。

例子：

```text
|n> = (1,0)
|psi> = (a,b)
```

则投影结果是：

```text
P_n|psi> = (a,0)
```

也就是只保留第一个方向的分量。

### 6.4 完备性：所有基底方向加起来覆盖整个空间

如果一组正交归一基底覆盖了整个空间，那么：

```text
sum_n |n><n| = I
```

这叫完备性。

普通语言：

```text
把所有基底方向的投影加起来，就等于什么都不丢地保留原向量。
```

对二维空间：

```text
|1><1| + |2><2| = I
```

所以：

```text
|psi> = I|psi> = |1><1|psi> + |2><2|psi>
```

这就是展开公式的算符写法。

### 6.5 从有限维到无限维

前面我们用二维、三维向量讲概念。量子力学中的位置空间通常是连续的，所以会出现无限维。

有限维展开：

```text
|psi> = sum_n c_n |n>
```

连续位置基底展开：

```text
|psi> = integral dx psi(x)|x>
```

有限维归一化：

```text
sum_n |c_n|^2 = 1
```

连续位置归一化：

```text
integral dx |psi(x)|^2 = 1
```

有限维内积：

```text
<phi|psi> = sum_n phi_n* psi_n
```

连续位置内积：

```text
<phi|psi> = integral dx phi*(x) psi(x)
```

你不需要害怕积分形式。它只是把“对所有分量求和”换成“对连续位置积分”。

### 最重要的概念关系

```text
态 -> 向量
表示方式 -> 基底下的坐标
概率 -> 振幅模平方
观测量 -> 厄米算符
可能结果 -> 本征值
特殊确定态 -> 本征向量
时间演化 -> 幺正变换
```

## 参考答案

### A

长度：

```text
||v|| = sqrt(2^2 + 2^2) = sqrt(8) = 2sqrt(2)
```

归一化：

```text
v_norm = (1/sqrt(2), 1/sqrt(2))
```

### B

先取第一个向量的复共轭：

```text
v* = (1, -i)
```

所以：

```text
<v|w> = 1 x 1 + (-i)(-i) = 1 - 1 = 0
<v|v> = 1 x 1 + (-i)i = 2
```

### C

```text
A(1,0) = 3(1,0)
A(0,1) = -2(0,1)
```

本征值为 `3` 和 `-2`，对应本征向量可取 `(1,0)` 和 `(0,1)`。

### D

对：

```text
u_+ = (1/sqrt(2))(1,1)
```

有：

```text
Bu_+ = u_+
```

本征值为 `1`。

对：

```text
u_- = (1/sqrt(2))(1,-1)
```

有：

```text
Bu_- = -u_-
```

本征值为 `-1`。

### E

投影到 `(1,0)` 方向，只保留第一个分量：

```text
P_n|psi> = (3/5, 0)
```

这表示态在 `|n>` 方向上的部分。对应概率是：

```text
|3/5|^2 = 9/25
```

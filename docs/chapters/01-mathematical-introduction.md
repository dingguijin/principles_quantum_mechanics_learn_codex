# 1. Mathematical Introduction

本章是整本书最重要的地基。量子力学把“状态”放进向量空间，把“可测量的物理量”表示成算符，把“测量结果”联系到本征值。先掌握这套语言，后面的薛定谔方程、角动量、自旋都会变得统一。

## 本章目标

学完后你应该能做到：

- 解释什么是向量空间、基底、维数。
- 计算内积、范数和正交归一。
- 理解线性算符、矩阵表示和基变换。
- 说清本征值和本征向量为什么重要。
- 初步理解厄米算符、幺正算符和投影算符。

## 1.1 向量空间

向量空间是一组对象的集合，这些对象可以相加，也可以乘以数，并且结果仍在集合里。

二维平面向量是最熟悉的例子：

```text
v = (3, 4)
w = (1, -2)
v + w = (4, 2)
2v = (6, 8)
```

量子力学中的“态”也被当作向量，但它们未必是箭头。它们可以是列向量，也可以是函数。例如一个粒子在一维空间中的波函数 `psi(x)`，也可以看作无限维空间中的一个向量。

## 1.2 线性组合、基底和坐标

如果有两个基向量：

```text
e1 = (1, 0)
e2 = (0, 1)
```

那么任意二维向量都可写成：

```text
v = v1 e1 + v2 e2
```

这里 `v1`、`v2` 是坐标。注意：向量本身和它在某个基底下的坐标不是同一件事。换一组基底，坐标会变，但向量代表的对象不变。

量子力学里这点非常关键。一个态可以在位置基底下写成 `psi(x)`，也可以在动量基底下写成 `phi(p)`。这是同一个态的两种表示。

## 1.3 内积和长度

内积把两个向量变成一个数。实二维向量的内积是：

```text
v · w = v1 w1 + v2 w2
```

复向量的内积要对第一个向量取复共轭：

```text
<v|w> = v1* w1 + v2* w2 + ...
```

其中 `*` 表示复共轭。

长度由内积定义：

```text
||v|| = sqrt(<v|v>)
```

量子态通常要求归一化：

```text
<psi|psi> = 1
```

物理解释：总概率为 1。

## 1.4 正交和归一

如果：

```text
<v|w> = 0
```

就说 `v` 和 `w` 正交。若一个向量长度为 1，就说它归一。

一组向量若两两正交且每个长度为 1，就是正交归一组。正交归一基底让展开特别简单：

```text
|psi> = sum_n c_n |n>
c_n = <n|psi>
```

物理上，`|c_n|^2` 常常对应在第 `n` 个基态上测量到相应结果的概率。

## 1.5 算符和矩阵

算符是作用在向量上的规则：

```text
A |v> = |w>
```

如果算符满足：

```text
A(a|v> + b|w>) = a A|v> + b A|w>
```

它就是线性的。量子力学主要使用线性算符。

在选定基底后，线性算符可以表示为矩阵。比如：

```text
A = [[1, 2],
     [3, 4]]
```

矩阵不是算符本身，而是算符在某组基底下的表示。换基底后矩阵会变，但算符对应的物理操作不变。

## 1.6 本征值和本征向量

如果某个非零向量 `|v>` 满足：

```text
A |v> = lambda |v>
```

那么 `|v>` 是 `A` 的本征向量，`lambda` 是本征值。

这表示：算符 `A` 作用后没有改变向量方向，只把它放大、缩小或乘上相位。

量子力学中，本征值的地位非常重要：一个可观测量的可能测量结果就是相应算符的本征值。

例子：

```text
A = [[2, 0],
     [0, 5]]
```

则：

```text
A (1, 0)^T = 2 (1, 0)^T
A (0, 1)^T = 5 (0, 1)^T
```

所以 `(1,0)^T` 和 `(0,1)^T` 是本征向量，`2` 和 `5` 是本征值。

## 1.7 厄米算符

厄米算符满足：

```text
A^\dagger = A
```

其中 `dagger` 表示先转置再复共轭。

厄米算符的两个关键性质：

- 本征值是实数。
- 不同本征值对应的本征向量正交。

这正好符合物理要求：测量结果必须是实数，彼此不同的测量结果应该能被清楚区分。因此，量子力学中可观测量通常由厄米算符表示。

## 1.8 幺正算符

幺正算符满足：

```text
U^\dagger U = I
```

它保持内积和长度：

```text
<Uv|Uw> = <v|w>
```

物理解释：系统随时间演化时，总概率不能凭空变多或变少，所以时间演化应保持态的归一化。这就是幺正性的意义。

## 1.9 投影

如果有归一向量 `|n>`，投影到该方向的算符可以写成：

```text
P_n = |n><n|
```

对任意态：

```text
P_n |psi> = |n><n|psi>
```

这里 `<n|psi>` 是态 `|psi>` 在 `|n>` 方向上的分量。投影算符是连接“态展开”和“测量概率”的桥。

## 1.10 从有限维到无限维

有限维中，我们用求和：

```text
|psi> = sum_n c_n |n>
```

连续空间中，求和变成积分：

```text
|psi> = integral dx psi(x) |x>
```

归一化条件也从：

```text
sum_n |c_n|^2 = 1
```

变成：

```text
integral dx |psi(x)|^2 = 1
```

这就是波函数形式和抽象向量形式的连接。

## 常见误区

- 不要把向量只理解成几何箭头；函数也可以是向量。
- 不要把矩阵当成本体；矩阵是算符在某个基底下的表示。
- 不要把波函数直接当概率；概率密度是模平方。
- 不要忽视复共轭；复内积少了共轭会破坏概率解释。

## 自测题

1. 为什么量子态需要归一化？
2. 一个向量换基底后，哪些东西变了，哪些东西没变？
3. 为什么可观测量要用厄米算符表示？
4. 幺正算符为什么适合描述时间演化？
5. 若 `|psi> = c1|1> + c2|2>` 且 `|1>`、`|2>` 正交归一，测量得到第 1 个基态对应结果的概率是多少？

## 建议练习

手算一个 `2 x 2` 厄米矩阵的本征值和本征向量。最简单可取：

```text
A = [[1, 0],
     [0, -1]]
```

然后再试：

```text
B = [[0, 1],
     [1, 0]]
```

你会发现这两个矩阵对应的本征基底不同。这是后面理解自旋和测量的核心。

# 13. The Hydrogen Atom

氢原子是量子力学解释原子光谱的经典成功案例。它把中心势、角动量和束缚态结合在一起。

## 本章目标

- 理解库仑势下的定态问题。
- 知道变量分离的作用。
- 理解主量子数、角量子数和磁量子数。
- 看出能级简并的来源。

## 核心讲解

氢原子近似为电子在质子库仑势中运动：

$$
V(r) = -k/r
$$

由于势能只依赖半径 `r`，系统具有转动对称性。问题可以分成径向部分和角向部分。角向部分由角动量理论处理，径向部分决定束缚态能量。

氢原子的能级离散，这解释了原子光谱中的特定谱线。

# 21. Path Integrals: Part II

本章继续发展路径积分，把它用于更复杂的量子系统和近似计算。它也是通向量子场论的重要入口。

## 本章目标

- 巩固传播子和作用量表述。
- 理解高斯积分在路径积分中的核心地位。
- 知道路径积分如何处理微扰。
- 看出路径积分与现代物理语言的联系。

## 核心讲解

路径积分的形式美在于它把动力学压缩进作用量：

$$
\text{amplitude} \sim \int \mathcal D[\text{path}]\,
\exp\!\left(\frac{iS[\text{path}]}{\hbar}\right)
$$

对于二次型作用量，路径积分往往可以精确计算或化成高斯积分。对于更复杂相互作用，则通常展开成微扰级数。

这一章的学习重点不是立刻掌握所有技巧，而是理解路径积分为何强大：它天然强调路径、相位、边界条件和对称性，在场论、统计物理和凝聚态中都非常有用。

# 0. 零基础准备

这份预备课不是量子力学正文，而是为了让你读第 1 章时不被术语挡住。你不需要先成为数学高手，但必须先学会几个基础词的意思：数、函数、向量、矩阵、导数、积分、概率、能量。

我会用“一对一上课”的方式讲。你每读一小节，都要停下来做后面的检查题。如果检查题说不出来，不要硬往后冲，因为后面只是把这些语言组合得更复杂。

## 0.1 先建立学习心态

量子力学刚开始最容易卡在两个地方。

第一，很多词看起来熟悉，比如“位置”“动量”“能量”，但在量子力学里它们会变成算符和测量结果。你不能只靠日常直觉。

第二，很多公式看起来抽象，比如 `|psi>`、`<phi|psi>`、`H|psi>`。这些符号不是为了显得高级，而是为了精确表达“系统处在什么状态”“我们问什么问题”“答案出现的概率是多少”。

所以我们采用一个固定学习流程：

1. 先问：这个符号代表什么对象？
2. 再问：它能和什么对象做运算？
3. 最后问：算出来的东西有什么物理含义？

比如：

```text
|psi>
```

不要急着读成“波函数”。先把它读成：

```text
系统当前的状态
```

然后再学它可以展开、归一化、被算符作用、用来计算概率。

## 0.2 你不需要一开始就会什么

你现在可以不会：

- 高等线性代数。
- 复杂微分方程。
- 抽象代数。
- 电磁学细节。
- 相对论细节。

但你要愿意反复练习：

- 把普通语言翻译成数学符号。
- 把数学符号翻译回普通语言。
- 对每个公式做简单数值例子。

量子力学最怕“看懂了但不会算”。如果一个公式不能拿简单数字算一遍，通常说明理解还不稳。

## 0.3 数：实数、复数和相位

### 实数

实数就是你熟悉的数轴上的数，比如：

```text
-2, 0, 3.5, pi
```

长度、质量、时间、能量的测量结果通常是实数。原因很朴素：实验仪器最后显示的读数必须是一个实数。

### 复数

复数写作：

```text
z = a + bi
```

其中：

```text
i^2 = -1
```

`a` 是实部，`b` 是虚部。比如：

```text
z = 3 + 4i
```

它的复共轭是：

```text
z* = 3 - 4i
```

它的模平方是：

```text
|z|^2 = z* z = (3 - 4i)(3 + 4i) = 25
```

注意这里得到的是实数非负数。量子力学中概率来自“模平方”，这就是为什么复数可以参与描述状态，而概率仍然是普通实数。

### 相位

复数也可以写成“大小 + 角度”的形式：

```text
z = r e^{i theta}
```

核心公式是：

```text
e^{i theta} = cos theta + i sin theta
```

这里的 `theta` 叫相位。量子力学中，相位会影响干涉。两个量子振幅相加时，不只是大小相加，相位也决定它们是增强还是抵消。

### 检查点

请你用自己的话回答：

1. 为什么 `3 + 4i` 不是概率，但 `|3 + 4i|^2` 可以变成非负实数？
2. 复共轭 `a + bi -> a - bi` 做了什么？
3. 相位为什么可能影响两个振幅相加的结果？

## 0.4 函数：把输入变成输出的规则

函数就是输入一个数，输出一个数的规则。

例子：

```text
f(x) = x^2
```

输入 `x = 3`：

```text
f(3) = 9
```

在物理中，函数常用来描述“某个量随另一个量变化”。比如：

```text
x(t)
```

表示位置随时间变化。

再比如：

```text
V(x)
```

表示位置 `x` 处的势能。

量子力学中最常见的函数是波函数：

```text
psi(x)
```

你可以先把它读成：

```text
在位置表示下，系统状态对每个位置 x 给出的复数振幅
```

不要把 `psi(x)` 直接读成概率。概率密度是：

```text
|psi(x)|^2
```

### 检查点

1. `x(t)` 和 `V(x)` 都是函数，但输入和输出分别是什么？
2. `psi(x)` 为什么可以是复数？
3. 为什么 `|psi(x)|^2` 比 `psi(x)` 更适合和概率联系起来？

## 0.5 向量：一组可以线性组合的对象

你熟悉的二维向量可以写成：

```text
v = (3, 4)
```

它不是两个互不相干的数，而是一个整体。你可以把它想成平面上的一个箭头，也可以把它想成“两个方向上的分量”。

两个向量可以相加：

```text
(3, 4) + (1, -2) = (4, 2)
```

向量可以乘以数：

```text
2(3, 4) = (6, 8)
```

只要某类对象可以做这两件事，并且结果仍属于同一类对象，我们就可以把它们当成向量空间里的向量。

这句话非常重要：函数也可以是向量。

比如函数 `f(x)` 和 `g(x)`，你可以相加：

```text
h(x) = f(x) + g(x)
```

也可以乘以数：

```text
k(x) = 2 f(x)
```

所以波函数也可以被当成向量。这就是为什么量子态可以统一写成：

```text
|psi>
```

它可能在某个表示下是列向量，也可能在位置表示下是函数。

### 最小例子

设：

```text
f(x) = x
g(x) = x^2
```

那么：

```text
h(x) = 2f(x) + 3g(x) = 2x + 3x^2
```

这就是函数的线性组合。

### 检查点

1. 为什么函数可以被看成向量？
2. `(3,4)` 这个向量和它的两个分量是什么关系？
3. `2f(x) + 3g(x)` 里面，“2”和“3”扮演什么角色？

## 0.6 矩阵：把向量变成另一个向量的机器

矩阵可以看成一种变换。

比如：

```text
A = [[2, 0],
     [0, 3]]
```

作用在：

```text
v = (1, 1)
```

上面，得到：

```text
Av = (2, 3)
```

这个矩阵把第一个方向拉长 2 倍，把第二个方向拉长 3 倍。

再看一个矩阵：

```text
B = [[0, 1],
     [1, 0]]
```

它作用在：

```text
(3, 4)
```

上，得到：

```text
(4, 3)
```

它交换两个分量。

量子力学中，“可观测量”会被表示成算符。有限维情况下，算符常常就是矩阵。矩阵不是一堆数而已，它告诉你“对状态做什么操作”。

### 检查点

1. 矩阵为什么可以被叫做“变换”？
2. 上面的矩阵 `B` 对向量做了什么？
3. 如果一个矩阵作用在向量后，只改变长度不改变方向，这种向量后面会叫什么？

## 0.7 导数：变化率

导数回答的问题是：

```text
一个量变化得有多快？
```

如果位置是时间的函数：

```text
x(t)
```

那么速度是：

```text
v(t) = dx/dt
```

这表示位置对时间的变化率。

例子：

```text
x(t) = t^2
```

那么：

```text
v(t) = dx/dt = 2t
```

当 `t = 3` 时：

```text
v(3) = 6
```

量子力学中导数会反复出现，因为动量算符在位置表示下和导数有关：

```text
p -> -i hbar d/dx
```

你现在不需要马上理解为什么，只要先知道：导数是描述变化的工具，而动量和空间变化有深刻关系。

### 检查点

1. `dx/dt` 的普通语言意思是什么？
2. 如果 `x(t) = 5t`，速度是多少？
3. 为什么“变化率”会和运动有关？

## 0.8 积分：累积

积分回答的问题是：

```text
把很多小贡献加起来，总共是多少？
```

如果速度恒定：

```text
v = 3
```

走 `2` 秒，位移是：

```text
3 x 2 = 6
```

如果速度随时间变化，就要把每一小段时间里的小位移加起来：

```text
位移 = integral v(t) dt
```

量子力学中，积分最常见的用途是把概率密度加起来。

如果 `|psi(x)|^2` 是概率密度，那么在区间 `[a,b]` 内找到粒子的概率是：

```text
P(a <= x <= b) = integral_a^b |psi(x)|^2 dx
```

总概率必须是 1：

```text
integral_all_space |psi(x)|^2 dx = 1
```

### 检查点

1. 积分为什么可以理解成“累积”？
2. 概率密度和概率有什么区别？
3. 为什么总概率必须归一化为 1？

## 0.9 概率：量子力学的输出语言

经典力学常常给确定答案：

```text
初始位置 + 初始速度 -> 未来位置
```

量子力学常常给概率答案：

```text
初始态 + 测量问题 -> 各结果的概率
```

概率必须满足：

```text
0 <= P <= 1
所有互斥结果的概率之和 = 1
```

如果一个系统只有两个可能结果，比如自旋向上和向下：

```text
P(up) = 0.7
P(down) = 0.3
```

这没有问题，因为它们加起来是 1。

如果写成：

```text
P(up) = 0.7
P(down) = 0.6
```

就不可能，因为总和超过 1。

量子力学中，概率通常来自振幅的模平方：

```text
probability = |amplitude|^2
```

这是贯穿全书的核心规则之一。

### 检查点

1. 概率为什么不能是负数？
2. 为什么互斥结果的概率要相加为 1？
3. “振幅”和“概率”为什么不是同一个东西？

## 0.10 最低物理语言

### 位置

位置告诉你物体在哪里。在一维中用 `x` 表示。

### 速度

速度告诉你位置变化得多快：

```text
v = dx/dt
```

### 动量

动量是：

```text
p = mv
```

它和运动状态有关。质量越大、速度越大，动量越大。

### 动能

非相对论动能是：

```text
K = p^2/(2m)
```

也可写成：

```text
K = (1/2)mv^2
```

### 势能

势能通常写成：

```text
V(x)
```

它表示不同位置对应不同能量。比如弹簧附近常见：

```text
V(x) = (1/2)kx^2
```

### 哈密顿量

哈密顿量通常表示总能量：

```text
H = K + V
```

在一维非相对论问题中：

```text
H = p^2/(2m) + V(x)
```

后面薛定谔方程会写成：

```text
i hbar d|psi>/dt = H|psi>
```

这句话的直觉是：哈密顿量决定量子态如何随时间演化。

### 检查点

1. `p = mv` 中，`m` 和 `v` 分别是什么？
2. `K = p^2/(2m)` 描述什么能量？
3. 为什么 `H` 会成为后面最重要的对象之一？

## 0.11 进入第 1 章前的最低要求

你不需要把上面内容背得滚瓜烂熟，但应该能做到：

1. 给出一个复数，算出它的复共轭和模平方。
2. 看懂函数 `f(x)` 是输入 `x` 输出 `f(x)`。
3. 对简单向量做加法和数乘。
4. 理解矩阵是作用在向量上的变换。
5. 说出导数是变化率，积分是累积。
6. 说出概率必须非负且总和为 1。
7. 知道 `|psi(x)|^2` 才和概率密度有关。

## 0.13 参考答案

### A

```text
z* = 1 + 2i
|z|^2 = (1 + 2i)(1 - 2i) = 5
```

### B

```text
f(0) = 1
f(2) = 5
f(-1) = 2
```

### C

```text
(2, -1) + (3, 4) = (5, 3)
3(2, -1) = (6, -3)
```

### D

```text
Av = (2, 10)
```

### E

```text
p = 6
K = 9
```

如果这些题都能独立完成，就进入第 1 章。若不能，先把卡住的题重做一遍；量子力学的难度不是从奇怪概念开始的，而是从这些基础对象没站稳开始的。

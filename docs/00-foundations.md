# 0. 零基础准备

这本书的开头已经有数学导论，但它默认你至少见过一点向量、矩阵、微积分和经典力学。真正从 0 开始时，先不要急着理解“量子到底是什么”，而要先学会量子力学使用的语言。

## 你需要先接受的观念

量子力学不是在日常直觉上加几个新词。它更像一套计算规则：给定系统状态，给定要问的问题，理论告诉你各种结果出现的概率。学习的第一目标不是想象电子“真实长什么样”，而是学会把问题翻译成数学对象。

## 最低数学包

### 1. 数和函数

你需要熟悉：

- 实数：长度、时间、能量通常用实数表示。
- 复数：量子态通常需要复数，写作 `z = a + bi`。
- 指数函数：`e^x`，以及复指数 `e^{i theta}`。
- 三角函数：`sin theta`、`cos theta`。

关键公式：

```text
e^{i theta} = cos theta + i sin theta
```

它说明复数可以表示“大小 + 相位”。量子力学里，相位不是装饰，而会影响干涉。

### 2. 向量和矩阵

普通二维向量可以写成：

```text
v = (v1, v2)
```

矩阵作用在向量上：

```text
A v = w
```

直观理解：矩阵是“把一个向量变成另一个向量”的机器。量子力学中的算符也是这种机器，只是空间可以更抽象，甚至无限维。

### 3. 微积分

需要知道：

- 导数描述变化率：速度是位置对时间的导数。
- 积分描述累积：概率密度积分得到总概率。
- 偏导数处理多变量函数。

最重要的物理直觉：

```text
速度 = 位置变化率
力 = 动量变化率
```

### 4. 概率

量子力学给出的常常不是确定答案，而是概率。概率必须满足：

```text
每个概率 >= 0
所有可能结果的概率之和 = 1
```

波函数本身一般不是概率，波函数的模平方才对应概率密度：

```text
P(x) dx = |psi(x)|^2 dx
```

## 最低物理包

### 1. 位置、速度、动量

在一维中：

```text
位置: x
速度: v = dx/dt
动量: p = m v
```

其中 `m` 是质量。

### 2. 能量

常见能量：

```text
动能: K = p^2 / (2m)
势能: V(x)
总能量: E = K + V
```

量子力学中，能量通常对应哈密顿算符 `H`。

### 3. 经典力学的核心问题

经典力学问：

```text
已知初始位置和速度，未来的位置和速度是什么？
```

量子力学问：

```text
已知系统状态，某次测量得到某个结果的概率是多少？
```

## 学习方法

每章学习时做三件事：

1. 记住对象：这一章引入了哪些数学对象？
2. 记住规则：这些对象之间如何计算？
3. 记住物理解释：计算结果对应什么可观测量？

不要把每个公式当成孤立结论。你要问：

- 这个公式输入什么？
- 输出什么？
- 单位是什么？
- 极限情况下是否回到熟悉结果？

## 本阶段自测

1. 为什么量子力学需要复数，而概率必须是实数非负？
2. 矩阵为什么可以被看作“变换”？
3. `|psi(x)|^2` 和 `psi(x)` 的物理含义有什么不同？
4. 动量 `p = mv` 和能量 `E = p^2/(2m) + V(x)` 分别描述什么？

如果这四个问题能用自己的话说清楚，就可以进入第一章。
